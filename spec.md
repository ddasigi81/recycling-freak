## 1. 개발환경
- HTML/CSS/JS로만 구현
- Design Theme: Clean, Eco, Green

## 2. 메인 화면 레이아웃 구조
1. **Header**: 중앙 상단에 6개 카테고리 가로 배열
   - 중소형 폐가전, 폐의약품, 의류, 폐형광등/폐건전지, 대형폐기물 수수료, 생활쓰레기 배출
2. **Search Bar**: 도로명/지번 선택  + 주소 입력 Input
3. **Main Content (2분할)**:
   - 좌측: 수거함 위치 정보 텍스트 형식으로 표시
   - 우측: 지도 영역 (추후 카카오맵 연동 예정, 현재는 Placeholder)

CSS 와 레이아웃을 절대 수정하지 않는다.
이제는 openapi 통해서 데이터만 연동하여, 화면에 표시해주는 것에 주력한다.
